    # 变量对象的定义

    变量对象是与执行上下文相关的数据作用域 ， 存储了上下文当中定义的变量和函数

    # 变量对象的理解

    变量对象分为全局上下文的变量对象和函数上下文的变量对象 ， 全局的变量对象就是 window ， 这个变量当中存储了一些预定义的变量 ， 函数和属性 ，函数上下文当中的变量对象(AO)存储的是 函数的形参 arguments ， 函数声明 ， 变量声明 。然后随着函数代码的执行 ， 这些变量对象当中的值也会进行修改

    // 函数代码还没有执行 ， 只是先进入了执行上下文
 AO = {
    arguments: {
         0: 1 ，
         length: 1
     } ，
     a: 1 ，
     b: undefined ，
     c: reference to function c(){} ，
     d: undefined
 }
 // 随着代码的执行
 AO = {
     arguments: {
         0: 1 ，
         length: 1
     } ，
     a: 1 ，
     b: 3 ，
     c: reference to function c(){} ，
     d: reference to FunctionExpression "d"
 }
